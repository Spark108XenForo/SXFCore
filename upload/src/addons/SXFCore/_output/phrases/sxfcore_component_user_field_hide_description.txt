Hiding the values of additional user fields in the profile. Each user can customize by editing his profile, above each additional field there is a dropdown list with a choice of display options: "Only me" - Where the value of the field sees only the user; "Subscribers only" - Where only the user and his / her subscribers see the value of the field; "Only authorized" - Where the value of the field can be seen by all authorized users; "All users" - Where the value of the field can be seen by absolutely all users, including unauthorized users. <span style = "color: red"> Note that the administrator always sees the value of the fields regardless of the user's settings. </ span> You can also configure via ACP.
